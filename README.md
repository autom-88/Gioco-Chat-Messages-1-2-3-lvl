# Gioco-Chat-Messages-1-2-3-lvl

Chat Secret Game è un progetto educativo sviluppato in HTML, CSS e JavaScript che introduce, in modo interattivo, concetti di logica, sviluppo web e cybersecurity attraverso una serie di livelli ispirati ai terminali Linux investigativi.

🎮 Caratteristiche

Terminale interattivo
Puzzle progressivi
Sistema di livelli
Comandi ispirati a Linux (help, ls, ls -a, ls -l, cat, pwd, whoami, clear)
File nascosti e indizi distribuiti
Effetti grafici in stile cyberpunk
Animazioni e transizioni
Codice JavaScript commentato e facilmente comprensibile
Nessuna libreria complessa richiesta

📚 Obiettivi didattici

Il progetto è stato realizzato per esercitarsi con:

HTML5
CSS3
JavaScript (ES6)
Manipolazione del DOM
Eventi
Local Storage
Gestione di stringhe e array
Logica di gioco
Interfacce utente interattive
Organizzazione del codice

🚀 Come giocare

Avvia index.html.
Esplora il terminale.
Utilizza i comandi disponibili.
Leggi i file e raccogli gli indizi.
Risolvi gli enigmi.
Sblocca il livello successivo.

⚠️ Nota

Questo progetto è stato creato esclusivamente a scopo educativo. Tutti i terminali, i file e gli scenari presenti nel gioco sono simulati e progettati per favorire l'apprendimento di HTML, CSS, JavaScript e del problem solving in un contesto ludico.
